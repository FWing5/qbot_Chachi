## 介绍
- 一个以ACG为主题的信息查询类机器人，目前功能有：
  - [x] 查询当日更新动画
  - [ ] 定时播报当日更新动画
  - [ ] 追踪bilibili【夏日幻听MCE】以获取最新动画视频资讯
  - [ ] 每日签到抽签
- 暂时不加入AI对话等功能


*茶知（Chachi），飞飞在冒险时路过森林遇到的女孩子，拥有一些来自森林的力量。*
